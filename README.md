# JobHunter AI

JobHunter AI is an open-source, AI-powered Chrome extension designed to supercharge your job search process. It automates tedious tasks with smart resume tailoring, ATS (Applicant Tracking System) scoring, and intelligent job notifications.

## Features

- **Resume Tailoring**: Automatically tailor your resume to match specific job descriptions using AI.
- **ATS Scoring**: Get real-time ATS compatibility scores for job postings.
- **Smart Notifications**: Stay updated with timely notifications about your job applications.
- **Broad Platform Support**: Seamless integration with major job boards including LinkedIn, Indeed, Glassdoor, and modern ATS platforms like Greenhouse, Ashby, and Lever.
- **Convenient Side Panel**: Access JobHunter AI tools directly alongside your current browsing tab.

## Installation

### Developer Mode (Local)

1. Clone the repository:
   ```bash
   git clone https://github.com/Hemanthsneu/jobhunter-ai.git
   ```
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top-right corner.
4. Click on **Load unpacked** and select the directory containing the cloned repository (where `manifest.json` is located).

## Usage

1. Navigate to a supported job board (e.g., LinkedIn Jobs, Indeed, Greenhouse).
2. Click the JobHunter AI extension icon in your browser toolbar or open the Side Panel.
3. Access AI tools to analyze the current job description, score your resume, and generate tailored application materials.

## Supported Platforms

- LinkedIn
- Indeed
- Glassdoor
- Greenhouse
- Ashby
- Lever

## Permissions Required

- **Storage**: Saving user preferences and resume data locally.
- **SidePanel**: Providing an accessible floating interface without leaving the job board.
- **ActiveTab & Tabs**: Interacting with job board pages to extract job details efficiently.
- **Notifications & Alarms**: Sending alerts about potential matches or follow-ups.
- **ContextMenus**: Adding quick actions to your right-click menu.
